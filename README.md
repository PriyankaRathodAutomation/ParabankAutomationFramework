# Parabank Automation Testing Framework

## Project Overview

This project is a complete Selenium Automation Testing Framework developed for the ParaBank web application using Java, Selenium WebDriver, TestNG, Maven, and Page Object Model (POM).

The framework is designed to automate end-to-end banking functionalities and demonstrate real-time automation testing concepts used in industry-level projects.
